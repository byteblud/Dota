# 🌱 BloomTrack AI

## Calm Productivity • Deep Focus • Real Progress

BloomTrack is an AI-powered productivity web application built with Flask.

Unlike traditional task apps, BloomTrack focuses on:

- 🌱 Real growth tracking
- 🧠 AI roadmap intelligence
- ⏳ Deep focus sessions
- 📊 Weekly analytics
- 🔥 Streak consistency
- 💌 AI reflections
- 📈 Beautiful progress visualization

---

# 🚀 Features

## 🌱 Task Garden
Tasks grow visually as you stay consistent.

## 🧠 AI Roadmap Tracking
Upload roadmap PDFs and track real completion.

## ⏳ Focus Mode
Minimal distraction-free productivity environment.

## 📊 Analytics Dashboard
Track consistency, focus time, and progress.

## 🔥 Streak System
Build momentum with weekly streak tracking.

# 🛠️ Tech Stack

- Flask

- Gemini API
- Chart.js
- PyMuPDF
- Render
